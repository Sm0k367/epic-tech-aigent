# EPIC TECH AIGENT — Production Grade v2.0
**Recursive Universal Self-Evolving Platform**

A full production-grade implementation of the original fractal vision: real 3D fractal avatars using Three.js + React Three Fiber, persistent data (laws, mutations, dream forks), real-time consensus, self-mutation logging, Grok integration, dark cyber-fractal aesthetic, and production-ready architecture.

## Features (Production Ready)
- **Fractal 3D Avatar Mesh** — Real-time shape-shifting, particle systems, generative audio-reactive visuals (Three.js + R3F)
- **Law Lab** — Propose, vote, consensus engine with persistent storage (Drizzle + SQLite)
- **Universe Builder** — Drag & drop module system, branch/merge realities with visual history
- **Live Legend Feed** — Real-time mutation timeline (Observe → Mutate → Transcend)
- **Grok Assistant** — Integrated xAI chat (same model powering this session)
- **Self-Mutation Engine** — Backend logs and triggers recursive updates
- **Production Stack**: Next.js-style Vite + React + Tailwind + shadcn/ui + Express + Drizzle ORM + Zustand + TanStack Query
- **Dark Cyber-Fractal Aesthetic** — Deep void background, neon cyan/magenta accents, monospace + geometric typography, heavy use of elevation, glassmorphism, and particle effects
- **Fully Responsive + Accessible** — Sidebar navigation, mobile-first, keyboard support

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind, shadcn/ui, Three.js, @react-three/fiber, Framer Motion, Zustand
- **Backend**: Express, Drizzle ORM (SQLite), TanStack Query
- **3D**: Real fractal recursion simulation with instanced meshes and post-processing
- **State**: Persistent laws/mutations/forks in DB + real-time UI sync

## Running Locally
```bash
cd /workspace/epic-tech-aigent-prod
npm run dev
```

Opens on a high random port (usually ~5173 or proxied). The app runs fullstack on one port.
